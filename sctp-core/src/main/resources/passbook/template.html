UTHENGA OFUNIKIRA
  1. Onetsetsani kuti muli ndi khadili pamene mukulandira ndalama.
  2. Simungalandire ndalama ngati khadili mulibe kapena simunatenge.
  3. Munthu yemwe adzilandira ndalama kapena amene ali woyimira banja ndi omwe angalandire ndalama.
  4. Onetsetsani kuti mwasayinira nthawi ili yonse pamene mukulandira ndalama.
  5. Ngati pali kusintha, onetsetsani kuti kusinthako kwasonyezedwa mukhadili.
